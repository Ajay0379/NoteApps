Steps to generate encrypted password.

1.	Place the jar locally on your machine.
2.	Run the jar file (command to execute jar from command prompt  java -jar <path where jar file is placed> <input password>)
	e.g. java -jar passwordGenerationApp.jar P@ssw0rd
3.	Console Output
		161ebd7d45089b3446ee4e0d86dbcf92
4. Run with invalid password so that password validation not passed
	Console Output
		FAILED
